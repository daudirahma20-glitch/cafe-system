# Cafe System
## Description
This is a web based cafe management system developed using HTML,CSS,HTML and MySQL.It helps to manage cafe operations such as customer records and order processing.
## Technologies used
PHP
HTML
CSS
MySQL
XAMPP
## Features
customer management
order processing
dashboard
## How to run
1.instal XAMPP
2.Copy project to htdocs
3.Start Apache and MySQL
4.Import cafe.sql into phpMyadmin
5.Open
http://localhost/cafe
